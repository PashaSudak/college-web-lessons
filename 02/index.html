Форматування

Теги для заголовків <h1> - <h6>

Опис:

HTML теги <h1>, <h2>, <h3>, <h4>, <h5>, <h6> - це шість заголовків показують відносну важливість блоку після заголовку в залежності від рівня заголовку h1, h2, h3, і т.д. h1 - заголовок першого рівня - найважливіший, а h6 - заголовок шостого рівня наймеш важливий. Теги заголовків є блоковими елементами. Текст, укладений всередині них, відображаються жирним шрифтом і мають різний розмір, відповідний до важливості заголовку. За замовчуванням, заголовок першого рівня <h1> відображається найбільшим шрифтом жирного накреселння, кожен наступний заголовок буде представлений розміром поменше.

Синтаксис:

<h1>Текст</h1>

Тег <strong>

Опис:

Тег <strong> - це тег який призначений для акцентування тексту. Браузери відображають такий текст жирним накресленням.

Синтаксис:

<strong>Текст</strong>

Тег <em>

Опис:

Тег <em> - це тег який призначений для акцентування текста.Браузери відображають такий текст курсивним шрифтом.

Синтаксис:

<em>текст</em>

Тег <small>

Опис:

Тег <small> - це тег зменшує розмір шрифту на одиницю в порівнянні зі звичайним текстом. В HTML розмір шрифту вимірюється в умовних одиницях від 1 до 7, середній розмір тексту, що використовується за замовчуванням, прийнятий 3.

Синтаксис:

<small>текст</small>

>

Тег <ins>

Опис:

Тег <ins> - призначений для виділення тексту, який був доданий до нової версії документа. Подібне форматування дозволяє відстежити, які зміни в тексті документа були зроблені.

Синтаксис:

<ins>текст</ins>

Тег <del>

Опис:

Тег <del> - використовується для виділення тексту, який був вилучений у новій версії документа. Подібне форматування дозволяє відстежити, які зміни в тексті документа були зроблені.

Синтаксис:

<del>текст</del>

Тег <sub>

Опис:

Тег <sub> - Відображає шрифт у вигляді нижнього індексу. Текст при цьому розташовується нижче базової лінії інших символів рядка і зменшеного розміру.

Синтаксис:

<sub>текст</sub>

Тег <sup>

Опис:

Тег <sup> - Відображає шрифт у вигляді верхнього індексу. Шрифт при цьому відображається вище базової лінії тексту і зменшеного розміру.

Синтаксис:

<sup>текст</sup>

Розмітка коду

Тег <code>

Опис:

Тег <sup> - призначений для відображення однієї або декількох рядків тексту, який являє собою програмний код. Сюди відносяться імена змінних, ключові слова, тексти функції і т.д.

Синтаксис:

<code>текст</code>

Тег <kbd>

Опис:

Тег <kbd> - використовується для позначення тексту, що набирається на клавіатурі або для назви клавіш.

Синтаксис:

<kbd>текст</kbd>

Тег <samp>

Опис:

Тег <samp> - використовується для відображення тексту, який є результатом виведення комп'ютерної програми або скрипта.

Синтаксис:

<samp>текст</samp>

Тег <pre>

Опис:

Тег <pre> - визначає блок попередньо форматованого тексту. Такий текст відображається зазвичай моноширінним шрифтом і з усіма пробілами між словами. За замовчуванням, будь-яку кількість прогалин йдуть в коді поспіль, на веб-сторінці показується як один

Синтаксис:

<pre>текст</pre>

Цитати та визначення

Тег <dfn>

Опис:

Тег <dfn> - Як правило, в документі, коли згадується новий термін, він виділяється курсивом і дається його визначення. При використанні цього терміна в подальшому, він вважається вже відомим читачеві. Тег <dfn> застосовується для виділення таких термінів при їх першій появі в тексті.

Синтаксис:

<dfn>текст<dfn>

Тег <abbr>

Опис:

Тег <abbr> - вказує, що послідовність символів є абревіатурою. За допомогою атрибута title дається розшифровка скорочення, що дозволяє розуміти абревіатуру тим людям, які з нею не знайомі. Крім того, пошукові системи індексують повнотекстовий варіант скорочення, що може використовуватися для підвищення рейтингу документа.

Синтаксис:

<abbr>текст</abbr>

Тег <q>

Опис:

Тег <q> - використовується для виділення в тексті цитат. Вміст контейнера автоматично відображається в браузері в лапках.

Синтаксис:

<q>текст</q>

Тег <cite>

Опис:

Тег <cite> - позначає текст як цитату або виноску на інший матеріал. Таке виділення зручно для зміни стилю тексту через CSS, а також застосовується для поділу коду HTML на структурні елементи

Синтаксис:

<cite>текст</cite>

Тег <blockquot>

Опис:

Тег <blockquot> - призначений для виділення довгих цитат всередині документа. Текст, позначений цим тегом, традиційно відображається як вирівняний блок з відступами зліва і справа (приблизно по 40 пікселів), а також з відбиття зверху і знизу.

Синтаксис:

<blockquot>текст</blockquot>

Розриви

Тег <br>

Опис:

Тег <br> - встановлює новий рядок в тому місці, де цей тег знаходиться.

Синтаксис:

<br>текст</br>

Тег <hr>

Опис:

Тег <hr> - Малює горизонтальну лінію, яка за своїм виглядом залежить від використовуваних параметрів, а також браузера.

Синтаксис:

<hr>текст</hr>

Списки

Тег <ul>

Опис:

Тег <ul> - встановлює маркований список. Кожен елемент списку повинен починатися з тега <li>.

Синтаксис:

<ul>
<li>елемент списку</li>
</ul>

Тег <ol>

Опис:

Тег <ol> - встановлює нумерований список. Кожен елемент списку повинен починатися з тега <li>.

Синтаксис:

<ol>
<li>елемент списку</li>
</ol>

Тег <li>

Опис:

Тег <li> - визначає окремий елемент списку.

Синтаксис:

<ol>
<li>елемент списку</li>
</ol>

Гіперпосилання

Тег <a>

Опис:

Тег <a> - є одним з важливих елементів HTML і призначений для створення посилань. Залежно від присутності атрибутів name або href тег <a> встановлює посилання або якір. Якорем називається закладка всередині сторінки, яку можна вказати в якості мети посилання. При використанні посилання, яка вказує на якір, відбувається перехід до закладки всередині веб-сторінки.

Синтаксис:

<a href="URL">...</a>
<a name="ідентифікатор">...</a>

Зображення

Тег <img>

Опис:

Тег <img> - призначений для відображення на веб-сторінці зображень в графічному форматі GIF, JPEG або PNG.

Синтаксис:

<img src="URL" alt="альтернативний текст">
